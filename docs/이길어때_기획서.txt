프로젝트명: CareRoute - 교통약자용 내비게이션 서비스
작성일: 2025-06-11

[1] 프로젝트 개요
------------------------
목적: 교통약자를 위한 안전하고 효율적인 경로 안내 웹서비스를 개발하여, 편의시설 정보와 사용자 맞춤 경로를 제공한다.
대상: 노약자, 임산부, 휠체어 사용자, 유모차 사용자 등 교통약자

[2] 핵심 기능
------------------------
- 회원 기능 (Auth)
  - 회원가입, 로그인, 회원정보 수정, 회원탈퇴

- 길 안내
  - 전동휠체어/도보 기준 경로 안내
  - 사용자 유형에 따라 필터링된 경로 제공
  - 실시간 편의시설 고장 시 우회 경로 제공

- 지도 마커 정보
  - 전동휠체어 충전소
  - 장애인 화장실
  - 차양막, 벤치
  - 무더위/한파 쉼터
  - 사용자 제보 기반 위험 요소 마커 (단차, 공사구간 등)

- 사용자 분류
  - 노약자/임산부: 급경사, 장거리 계단 제외
  - 휠체어/유모차 사용자: 계단 제거, 엘리베이터 필수
  - 환승 구간: 텍스트 기반 상세 경로 제공
  - 지하철 편의시설 고장 시 우회 로직 포함

- 제보 시스템
  - 사용자 제보 기능 (공사구간, 고장 등)
  - 관리자 페이지에서 제보 승인 및 관리

- 즐겨찾기 기능 [추가 기능]
  - 사용자가 자주 이용하는 장소를 즐겨찾기에 등록
  - 메인화면에서 즐겨찾기 기반 빠른 경로 안내 제공

- 음성 안내 (향후 추가 예정)
  - 길안내 정보에 대해 TTS 기반 음성 제공
  - 시각장애인 또는 화면 주시 어려운 사용자 지원

[3] 기술 환경
------------------------
- 프론트엔드: 반응형 웹 (모바일 중심), HTML/CSS/JS, 지도 API 연동
- 백엔드: Node.js (Express), RESTful API 설계
- DB: MySQL (ERD 기반 모델 설계)
- 개발도구: Visual Studio Code
- 인증: JWT 기반 사용자 인증

[4] 팀 구성
------------------------
- HTML/CSS (JSP): 이찬환
- JS (JSP): 최리준
- CONTROLLER/UTIL/FILTER(CONFIG): 한상원
- SERVICE: 김하경
- DAO/DTO(CONFIG): 이솔비
- 기획 및 전체 흐름: 이광주

[5] 제안 기능 및 향후 발전 방향
------------------------
- 다국어 지원 (외국인 교통약자 포함)
- 오프라인 모드 (저장된 경로 기반 제한적 안내)
- IoT 및 Beacon 연동 (실내 정밀 위치 안내)
- 실시간 사용자 위험 경고 기능
